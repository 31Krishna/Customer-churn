# 📊 Customer Churn Analysis Dashboard 
## 🚀 Project Overview

This project analyzes customer churn behavior using a telecom dataset to identify key factors influencing customer retention.
The dashboard provides both **high-level KPIs** and **deep analytical insights** to support business decision-making.

---

## 🎯 Objectives

* Analyze customer churn patterns
* Identify key drivers of churn
* Provide actionable insights to improve customer retention
* Visualize data using an interactive Power BI dashboard

---

## 📁 Dataset

* Source: IBM Telco Customer Churn Dataset
* Records: ~7,000 customers
* Features include:

  * Customer demographics
  * Account information (tenure, contract type)
  * Services (internet, payment method)
  * Billing details (monthly charges, total charges)
  * Churn status

---

## 🛠️ Tools & Technologies

* Power BI
* Data Cleaning (Power Query)
* DAX (Data Analysis Expressions)

---

## 📊 Dashboard Features

### 🔹 Page 1: Overview

* Total Customers
* Churn Customers
* Churn Rate (%)
* Revenue Lost
* Average Monthly Charges

**Visuals:**

* Churn Distribution
* Churn by Contract Type
* Churn by Payment Method

---

### 🔹 Page 2: Deep Analysis

* Churn by Tenure (binned analysis)
* Churn by Monthly Charges
* Churn by Internet Service

---

## 📈 Key Insights

* Customers with **month-to-month contracts** have the highest churn rate
* **Early tenure customers (0–12 months)** are more likely to churn
* Higher monthly charges are associated with increased churn
* **Fiber optic users** exhibit higher churn compared to DSL users
* Electronic check users contribute significantly to churn

---

## 💡 Business Recommendations

* Improve onboarding experience for new customers
* Introduce incentives for long-term contracts
* Re-evaluate pricing strategies for high-paying customers
* Enhance service quality for fiber optic users
* Promote auto-payment methods to reduce churn
